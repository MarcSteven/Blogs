The issue of installing cocoa pods

Actually cocoapods is a great tool to manage the third-party libraries in your project.If you wanna use more libraries,using it can improve your effective and save time.

How to install cocoapods ?

Firstly you must confirm whether your Mac own the environment of ruby.Ruby is very fashion recently….Very useful.Open your shell and then “ruby -version”,you can know the version of your Mac.

Now let’s start.

“gem install cocoapods” and then press enter ,you will see the successful info about it.

But I met a weird issue several days earlier.You don’t have write permission .If you meet this issue,pls do these step as below under your shell

1,brew update

2,brew install ruby

3,gem install cocoapods

Actually Mac offer you the ruby,but if you don’t install the environment of ruby yourself ,you cannot solve this issue.Hope this is useful for your issue.
