Build speed in Xcode

If you wanna show the build time and then show this Hub information in Xcode.

Just do as this simple command line .

Open your iterm tool ,and then input the order as below

defaults write com.apple.dt.Xcode ShowBuildOperationDuration YES

And then you can see the build time .

So simple and then you can learn more from here .
